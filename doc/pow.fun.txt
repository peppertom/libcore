@name pow
@title pow
@summary Raise a value to a power

@description 

Returns the value raised to the power. The caret character, `^`, is used as an alias for `pow` e.g. `x ^ y` is equivalent to `pow(x, y)`.

@param {number} value Base value to be raised
@param {number} exponent Exponent
@return {number} Vakue raised to the power of the exponent

@example pow(x, y)
@example x ^ y
